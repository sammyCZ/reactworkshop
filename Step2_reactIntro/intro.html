Introduction to using React

Step 1: Open Project Folder

  1. Open your code editor (VS Code) or GitHub 
    On VS Code: File -> Open -> Find the folder you just created during the Pre-Workshop Installation Guide

Step 2: Observation over React

  1. Open the index.html file 
    Inside the public folder, open the index.html file.
    React is called a single-page application.
In other words, React is loading the entire app on this index.html page.
    Look for the line <div id="root></div>.
    This is line is where all the feature code is put together and replaces this <div id="root></div>
line with your whole app.
  2. Understand Root Rendering 
    Inside the src folder, open the index.js file.
Look for the line: const root = ReactDom.createRoot(document.getElementbyId(‘root’));
(Line 7 on VS Code.)
    This line tells the computer to look for the documents element called "root"
    Look at the next 4 lines: root.render{...
    Here, we are declaring the root, which will be rendering the app.
    Inside the src folder, open the App.js.
Look at the return function. We can see that App.js is returning html code to ‘index.html’.
    What does this mean? In essence, React is really a fancy way to return html code! This means that you can add 
different components and styling pages i.e. css files to your project and they should all be 
rendered together thanks to your ‘index.html’ file.

Step 3: Launch the App

  1. Open a New Terminal 
    On VS Code: Terminal -> New Terminal.
On GitHub: Ctrl+Shift+`
    Navigate to your my-app folder using the command: cd my-app
    Launch the app using the command: npm start

Step 4: Customize the App

  1. Make a New Header 
    Inside the App.js file, look at the return function. 
In the return function, create a header tag anywhere in the page using the following JavaScript code: <h1>Your text here</h1>
  2. Add a Picture 
    Look up a picture you want to add to your App. 
Copy the image address. We’re going to add this underneath the header we just made.
    Under the header, add the Javascript code:

<img
src="link of image address here"
alt="Text shown if image fails to load"
style={{ 
	width: "100%", 
	maxWidth: "600px",
	height: "auto"
}} />
  3. Change the Background Color 
    Inside the src folder, go to the App.css file. 
    App.css is our styling page.
                                
In React, a .css file is used to define styles (like colors, fonts, and layout), which controls how your components look.
You can create your own .css file and import it into your component to apply those styles using class names.
This keeps your styling separate from your JavaScript logic, making your code cleaner and easier 
to manage.
    Go to the .App-header class.
Delete the default background color and insert your color of choice.

Tip: You can type the color or look for the hex value on Google.
