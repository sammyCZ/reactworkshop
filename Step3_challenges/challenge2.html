Challenge 2: Dynamic Color-Changing Background

  1. Goal: Let the user click a button that randomly changes the background color of the page.
  2. Navigate to your React project 
    cd my-app
  3. Create the component files 
    Inside the src folder, create a folder called components.
Inside it, create two files:
1. ColorChanger.js
2. ColorChanger.css
  4. Add this code to ColorChanger.js 
    // src/components/ColorChanger.js
import React, { useState } from "react";
import "./ColorChanger.css"; // Optional CSS file

const getRandomColor = () => {
  // Generate a random hex color
  const letters = "0123456789ABCDEF";
  let color = "#";
  for (let i = 0; i < 6; i++) {
    color += letters[Math.floor(Math.random() * 16)];
  }
  return color;
};

const ColorChanger = () => {
  const [bgColor, setBgColor] = useState("#ffffff"); // Default background

  const changeColor = () => {
    const newColor = getRandomColor();
    setBgColor(newColor); // Update state, triggers re-render
  };

  return (
    <div className="color-container" style={{ backgroundColor: bgColor }}>
      <h1>Background Color: {bgColor}</h1>
      <button onClick={changeColor}>Change Color</button>
    </div>
  );
};

export default ColorChanger;
  5. Add this code to ColorChanger.css 
    .color-container {
  height: 100vh;
  display: flex;
  flex-direction: column;
  justify-content: center;
  align-items: center;
  transition: background-color 0.5s ease;
}

button {
  padding: 10px 20px;
  font-size: 1.2rem;
  border: none;
  background-color: black;
  color: white;
  cursor: pointer;
  margin-top: 20px;
  border-radius: 8px;
  transition: transform 0.2s ease;
}

button:hover {
  transform: scale(1.05);
}
  6. Replace App.js with the following: 
    // src/App.js
import React from "react";
import ColorChanger from "./components/ColorChanger";

function App() {
  return <ColorChanger />;
}

export default App;
  7. Run the app 
    npm start
    Your React app should open automatically at http://localhost:3000.
  8. Challenge questions:
    • What else can you do to customize your background?
    • If you add text, can you change the text color based on the background?
  9. Close the app 
    Press Ctrl + C on Mac/Linux/Windows or stop the process in GitHub Codespaces.
