Challenge 1: Embed a Video

Goal: Play an external video (URL) inside your React App.
  1. Navigate to your React project 
    cd my-app
  2. Create a components folder 
    Inside the src folder, create a folder called components
  3. Create two files inside components: 
    VideoPlayer.js
VideoPlayer.css
  4. Add this code to VideoPlayer.js 
    // src/components/VideoPlayer.js
const VideoPlayer = () => {
  return (
    <div className="video-container">
      <iframe
        width="720"
        height="405"
        src="video link"
        title="YouTube video player"
        frameBorder="0"
        allow="autoplay; encrypted-media"
        allowFullScreen
      ></iframe>
    </div>
  );
};

export default VideoPlayer;
    Replace "video link" with the URL of the YouTube or external video you want to use.
  5. Add styling in VideoPlayer.css 
    .video-container {
  display: flex;
  justify-content: center;
  align-items: center;
  height: 100vh;
  background-color: #111;
}

video {
  border-radius: 12px;
  box-shadow: 0 8px 24px rgba(0, 0, 0, 0.3);
}
  6. Include your VideoPlayer into App.js 
    // src/App.js
import React from "react";
import VideoPlayer from "./components/VideoPlayer";

function App() {
  return <VideoPlayer />;
}

export default App;
    Comment out or delete your previous App.js before adding this code.
  7. Run your app 
    npm start
    If it’s not running already, this command will start the development server and open your app in the browser.
  8. Customize your background
    Try modifying the background-color in VideoPlayer.css or experiment with gradients for a unique style!
  9. Stop your app 
    Ctrl + C
    Works on Mac, Linux, Windows, or GitHub Codespaces.
