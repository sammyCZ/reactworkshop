Pre-Workshop Installation Guide

Steps for macOS, Windows, and GitHub Codespaces

Set Up React Using GitHub Codespaces

Step 1: Set Up GitHub Codespaces

  1. Open your Codespace 
    1. Go to your repository on GitHub
2. Click the green “Code” button and select “Open with Codespaces”
3. Wait for the Codespace environment to load
  2. Install Node.js (if missing)
    Run these commands one at a time in your terminal:
    sudo apt update
sudo apt install nodejs npm -y
    This installs Node.js and npm using the Ubuntu package manager.
  3. Verify installation
    Run these commands one at a time in your terminal:
    node -v
npm -v
    Each command should print a version number, confirming installation.

Step 2: Create React App using Node.js and npm

  1. Open Terminal 
    1. Press Ctrl+Shift+P (Windows/Linux) or Cmd+Shift+P (macOS) to open the Command Palette
2. Type "Terminal" and select "Terminal: Create New Terminal" from the suggestions
3. Press Return or Enter
  2. Create a new React project 
    npx create-react-app my-app
    This uses npm’s npx tool to set up React with all dependencies.
  3. Navigate to the project directory 
    cd my-app
  4. Start the development server 
    npm start
    Your React app should open automatically in the browser at http://localhost:3000.

You're set! Thank you for following the Pre-Workshop Installation Guide.

Set Up React Locally on Your Device

Step 1: Install Visual Studio Code

  1. Download VS Code 
    Go to https://code.visualstudio.com
    Choose the version for your operating system (macOS, Windows, or Linux).
  2. Install and open VS Code 
    Follow the setup wizard and open Visual Studio Code once complete.
  3. Verify installation 
    code --version
    If this prints a version number, VS Code is installed correctly.

Step 2: Install Node.js

macOS

  1. Open Terminal 
    1. Cmd+Space
2. Type "Terminal"
3. Press return
  2. Install Homebrew
    Copy the following command into the terminal and press enter.
    /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
    After installation, run brew --version to verify.
  3. Install Node.js
    Copy the following command into the terminal and press enter.
    brew install node
    This installs Node.js and npm.
  4. Verify installation
    Run these commands one at a time in your terminal:
    node -v
npm -v
    Each command should print the version you have, confirming installation.

Windows

  1. Official installer
    Watch the walkthrough video: View on YouTube 
    1. Visit https://nodejs.org
2. Download the LTS Windows Installer (.msi)
3. Run the installer and follow the prompts
    This installs Node.js and npm system-wide.
  2. Open Command Prompt 
    1. Click on the Windows Start menu (Windows Logo)
2. Type "Command Prompt" in the search bar and select Command Prompt
  3. Verify installation
    Run these commands one at a time in your terminal:
    node -v
npm -v
    Each command should print the version you have, confirming installation.

Step 3: Install React using Node.js and npm

  1. Make project folder 
    1. Create a project folder somewhere in your file management app
2. Go to VS Code
3. Navigate to File > Open Folder... and open your project folder
4. Navigate to Terminal > New Terminal
    Tip: You can use the pwd command in your terminal to verify your path.
  2. Create a new React project
    Copy the following command into the terminal and press enter
    npx create-react-app my-app
    This uses npm’s npx tool to set up React with all dependencies.
  3. Navigate into the my-app folder
    Copy the following command into the terminal and press enter
    cd my-app
  4. Launch your app! Start the development server
    Copy the following command into the terminal and press enter
    npm start
    Your React app should open automatically in the browser at http://localhost:3000.

You're set! Thank you for following the Pre-Workshop Installation Guide.
